# Golb

A simple blog using Laravel 8.

## Environment
- Laravel `8.4`
- composer `2.1`
- PHP `7.4`
- MySQL `8.0`

## Features
- Guest/User can:
    - List posts
    - Read posts
    - Register
- User can:
    - Login
    - Logout
    - List users
    - Delete users
    - Create posts
    - Edit posts
    - Delete posts
